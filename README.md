# Juego_en_7_dias
Cable Runner

Cable Runner es un videojuego de plataformas 2D desarrollado en Python y Pygame por estudiantes de Ingeniería en Software.
El jugador controla a un técnico que debe ascender por un datacenter cyberpunk utilizando un cable de conexión como mecánica principal. Para completar cada nivel deberá superar plataformas, trampas y obstáculos mientras mantiene conectado su cable hasta llegar al rack objetivo
